关于我的碎碎念

这个博客是自己在9102年年底心血来潮使用Hexo框架 + Github Pages搭建出来的，然后用的是hexo-theme-cactus这个主题，很方便、快速的就搭起来了（其实还有其他的框架，懒得去了解了）。第一次干，总想把它弄得花里胡哨、功能齐全，中途还有一次玩坏的时候，其实自己也知道内容才是博客的精华，但是我就是注重形式，略略略。当然，本博客中这些花里胡哨的功能，大多数都能在这个hexo搭建的博客 - 优化篇文章中找到，内容不是重点，好看才是王道==。

也是因为为了把这个博客做好，就像和孩子一样，想一步步把他完善，孕育他功能强大···，吼吼，才激起了我想学习前端的冲动（我本是个渣渣底层开发者）！

平时兴趣爱好也很广泛，喜欢日语不是二次元，喜欢学唱日语歌，想去日本旅游，喜欢代码但是自己的代码能力很弱很弱，喜欢用代码搞一下花里胡哨的东西自娱自乐，喜欢看恐怖篇、悬疑片，喜欢看世间的酸甜苦辣，喜欢狗狗，啥狗都喜欢，以后目测是个狗奴，嗯。平常喜欢玩知乎，乱七八糟的问题也在知乎上搜索，微博最多看看热搜平时也玩得少，口水话比较多，不怎么正经。喜欢折腾，人生就在于折腾，不想一辈子庸庸散散，碌碌无为，想搞点事情，至少要让自己的人生有意义，也不枉来世间走一趟了。

人生信条，感兴趣的可以在此篇文章life credo中看一下，热烈欢迎与信仰、理念相投的有缘人产生精神的共鸣。脾气暴躁，非礼勿扰。
